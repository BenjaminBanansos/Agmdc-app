# AGMDC Web Application

This project contains two main components:
1. **Public Website Replica**: An implementation of the `agmdc.in` home page.
2. **Organizational Web App**: An admin dashboard for managing the organization's pipeline.

## Tech Stack
- **HTML5**
- **Vanilla CSS3** (Custom properties, Flexbox/Grid)
- **Vanilla JavaScript** (ES6+)

## Project Structure
```
agmdc-app/
├── index.html          # Public Website Home (Replica)
├── admin.html          # Admin Dashboard (Organizational App)
├── assets/
│   ├── css/
│   │   └── style.css   # Main stylesheet
│   └── js/
│       └── app.js      # Main logic
```

## How to Run
Since this is a static site, you do not need any build tools or servers.
1. Navigate to the `agmdc-app` folder.
2. Double-click `index.html` to view the **Public Website**.
3. Double-click `admin.html` to view the **Admin Dashboard**.
